Welcome to Pune.io

A "smart" city data monitoring platform.
Here you can access all of your city's data and use it as yours!!!
Now's your chance to know your city, Pune, better, right from your fingertips!!!

We collect data from different devices spread across Pune.
You can access this data by using our API and build your very own Pune app.
Or you could simply look at the representation of that very data in our website.
The data can be anything from temperature or humidity to something you want.

Geo-Tagged

The data is geo-tagged by using GPS co-ordinates so that you can know for sure that you are accessing the data from a particular area only.

Powered by the Sun

The devices which are placed across Pune are powered by the amazing and powerful friend of ours, The Sun. This means you'll never run out of juice for your device.

Connected by GSM

The devices are always connected to the internet by the use of GSM. This means the devices are always online and will be uploading the data and other parameters such as battery level etc.

Crowd Sourced Data

The data will be sourced from the crowd. More than one device will be placed in a certain area so that the reliability of data will be high.

Stats for Nerds

Software

This is the list of software magic we are using

  • RESTful API
  • Leaflet.js for displaying Maps
  • Geo-Json output format
  • Node.js based server for serving the API requests

Hardware

This is the hardware we are using

  • LM35 for temperature sensing
  • SYHS220 for humidity
  • LDR for ambient light
  • PIR sensor module for motion sensing
  • ADXL335 accelerometer module for acceleration sensing of the device
  • Adafruit FONA GSM module for GSM connectivity
  • Adafruit GPS module for GPS locating
  • Atmega328p microcontroller
  • LTC3115-1 for powering up the microcontroller and everything else
  • MCP73871 for charging the battery using Solar panel
  • DS1307 RTC IC for timekeeping

Amazing things you can do with it

A List of magic tricks you can perform

  • Accessing the data from anywhere and on any device
  • Create a heatmap or heat timeline of any area or the whole city
  • Real time monitoring of City traffic and alerting yourself and the community of any traffic congestions
  • Providing advanced navigation based on GPS and Accelerometer which detects bumps, speedbreakers etc.
  • Any other application made for your mobile phone which can access all the features right in your hand
